# Outlast-the-Baddies-Avoid-the-Abyss
Polymorphic grid-based game where Hero (H) avoids Abyss (#) and Baddies while reaching Escape Ladder (*). Features Monsters (m/M) with 1/2-step chasing, Bats (~) seeking same column. Uses Grid template, BoardCell inheritance, try/catch for collision handling. Valgrind-checked memory management.
